# 🚀 CampusMart

A full-stack campus marketplace where students can buy, sell, and chat for products.

## ✨ Features
- 🔐 Authentication (Login/Register)
- 📦 Add / Edit / Delete Products
- 💬 Real-time Chat System
- 🧑‍💼 User Profile
- 📊 Dashboard (Sold / Available)
- 🔍 Search & Filter

## 🛠️ Tech Stack
- Frontend: React / Next.js + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT
